# WantedPushCrawler

###### 這是專為 PTT 爬蟲開發者所開發的函式庫 [PTTCrawlerLibrary](https://github.com/Truth0906/PTTCrawlerLibrary) 的範例程式
###### 會針對汪梯版每篇新文章的發文類型隨機從推文庫推文
###### 如果文章中有禁止推文的情況，程式可成功偵測並略過該篇文章

###### 執行檔可直接下載 WantedPushCrawler.zip
